# Sync and Offline Access

Nimbus Notes syncs your notes across devices automatically over an encrypted
connection. Sync usually completes within a few seconds of an edit. A small cloud
icon in the toolbar shows the current sync status.

If sync stalls, sign out and back in, then trigger a manual sync from
**Settings → Sync → Sync now**. Offline edits are queued locally and pushed when
you reconnect. Conflicting edits to the same note create a *conflict copy* rather
than overwriting your work, so nothing is lost.
