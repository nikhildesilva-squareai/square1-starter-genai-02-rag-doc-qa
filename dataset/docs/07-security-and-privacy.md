# Security and Privacy

Notes are encrypted in transit (TLS) and at rest (AES-256). Nimbus Notes
supports two-factor authentication (2FA) via an authenticator app; enable it in
**Settings → Security**. Team-plan admins can require 2FA for all members.

We never sell your data or use note contents to train advertising models. You can
permanently delete your account and all associated notes from
**Settings → Security → Delete account**; deletion is irreversible and completes
within 30 days across all backups.
