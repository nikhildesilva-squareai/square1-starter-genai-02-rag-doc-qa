# Import and Export

Nimbus Notes can import from Evernote (`.enex`), Markdown (`.md`), and plain text.
Use **Settings → Import** and select the files or a folder; imported notes keep
their titles, tags, and attachments where possible.

To export, open a note or notebook and choose **Export**. Supported export
formats are **PDF**, **Markdown**, and **HTML**. Exporting a whole notebook
produces a ZIP archive with one file per note. Exports never include other
people's private notebooks — only content you own or can edit.
