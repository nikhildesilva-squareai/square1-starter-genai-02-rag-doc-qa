# Getting Started with Nimbus Notes

Nimbus Notes is a note-taking app for teams. To create your first notebook,
click **New Notebook** in the top-left sidebar, give it a name, and press Enter.
Notes live inside notebooks; drag a note between notebooks to move it.

New accounts start on the **Free** plan with up to 3 notebooks. You can rename a
notebook at any time by double-clicking its title. Notebooks can be nested two
levels deep to keep large workspaces organised.
