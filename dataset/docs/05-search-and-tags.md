# Search and Tags

Press **Ctrl+K** (Cmd+K on Mac) to open search. Search matches note titles, body
text, and tags. Add a tag by typing `#` followed by the tag name anywhere in a
note; tags appear in the left sidebar for one-click filtering.

You can combine filters: for example, search `budget #finance` to find notes that
contain the word "budget" and carry the `finance` tag. Search is full-text and
updated within seconds of an edit, so newly written notes are findable almost
immediately.
